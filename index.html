Git - система контроля версий.

Для начала работы, необходимо скачать и установить соответствующий системе ПК дистрибутив, на странице /download
Для проверки корректности установки необходимо вызвать консоль на ПК и ввести команду: git -- version
Если выдается информация по данной команде, то Git установлен корректно.
